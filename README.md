# SMSLogin
Android native project to demonstrate sms sending and otp received functionality.

### Concepts Used
1. Singleton Class
2. Shared Preference - key value patetrn
3. Gradient Shape
4. Send sms from phone
5. Broadcast Receiver receive message received
6. Otp view
